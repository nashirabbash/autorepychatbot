## kalau anon muncul pesan kaya gini, berarti pasangan/stranger telah ditemukan/match:

```txt
Pasangan telah ditemukan!
Katakan, Hai untuk membalas pasangan anda 😊

/next - Untuk mencari obrolan baru
/stop - Untuk berhenti dari obrolan

🔊 PROMOTE 🔊

mimin bikin grup baru sini cari temen baruu 👉 @caripacar_teman_jodoh
```

itu atau kalau ga ini

```txt
Partner found 😺

/next — find a new partner
/stop — stop this chat

https://t.me/chatbot
```

## kalau anon muncul pesan kaya gini, berarti sedang mencari pasangan/stranger/belum match:

```txt
Sedang mencari obrolan...
```

atau kalau ga ini:

```txt
Looking for a partner...
```

## kalau anon muncul pesan kaya gini, berarti lawan bicara anda stop percakapan, atau bahkan skip percakapan, jadi langkah bot otomatis selanjutnya yaitu /search:

```txt
Lawan bicara telah meninggalkan percakapan!
Apakah Anda Ingin mengobrol dengan orang lain?

Ketik /search untuk mencari obrolan baru
```

atau kalau ga ini:

```txt
Your partner has stopped the chat 😞
Type /search to find a new partner

https://t.me/chatbot
```

## kalau anon muncul pesan kaya gini, berarti kamu yang mengehentikan percakapan:

```txt
You stopped the chat. Searching for a new partner...
```

atau kalau ga ini

```txt
Anda telah meninggalkan percakapan!
Apakah Anda Ingin mengobrol dengan orang lain?

Ketik /search untuk mencari obrolan baru
```

## kalau anon muncul pesan kaya gini, kalau muncul nya setelah anda send /next, abaikan. kalau munculnya setelah partner stop percakapan, berarti selanjutnya anda send /search

```txt
Jika mau, silakan tinggalkan feedback tentang pasangan Anda. Ini akan membantu kami mencarikan pasangan yang lebih sesuai untuk Anda di masa depan
```
